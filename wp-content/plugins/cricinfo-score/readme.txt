== Description ==

Cricket is the second most popular game in the world! And, it is the top most popular game in India. More than 65% of cricket fans/viewers are from India. This plug in is developed for them who loves to be updated when working. Enjoy and feel free to copy and share this with others.

== Installation ==

This plugin can be installed as a normal plugin.

To install as a plugin manually:

1. Download the ZIP file of the current version
2. Unzip the file on your computer
3. Upload the cricinfo-score folder to /wp-content/plugins

To install as a plugin automatically:

1. Visit Plugins -> Add New in your Site Admin (for normal WordPress installations)
2. Search for CricInfo
3. Click the "Install" link for this plugin

To activate the plugin on a single site:

1. Go to the Plugins menu within the Site Admin area and click the "Activate" link
2. Then go to Appearance->Widgets
3. You will find CricInfo widget.
4. Just drag and drop to sidebar.

That's it!
Enjoy!!
